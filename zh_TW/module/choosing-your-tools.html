選擇您的工具

所有的數位工具，無論是硬體還是軟體，都應該是安全的。也就是說，它們應該保護您免受監視，並能阻止您的裝置被人控制。可惜的是，目前情況並非如此。對於許多數位活動，您可能最終需要專用程式或專為提供特定安全功能的設備。本指引中使用的範例包括讓您加密訊息或檔案的軟體，如 PGP。

但是，有許多公司和網站提供安全程式或硬體，您要如何選擇適合的？

安全是一個過程，而不是單次購買 Anchor link

在改變您使用的軟體或購買新工具之前，首先要記住的是，沒有任何工具能夠提供絕對的監控防護。使用加密軟體通常會使其他人更難以閱讀您的通訊或檢視您電腦的檔案。但是，對您的數位安全進行的攻擊總是會找出您安全做法中最薄弱的部分。使用新的安全工具時，您應該考慮如何使用它可能會影響別人可以定位您的其他方式。例如，如果您決定使用加密傳訊程式與某個聯絡人交談，因為您知道手機可能遭到竊聽，那麼您使用此程式可能會給予對手線索，得知您在談論私人資訊？

其次，記住您的威脅模型。如果您的最大威脅是來自無法使用網路監控工具、私家偵探的實體監控，就不需要購買一些昂貴、聲稱是「防國安局」的加密電話系統。另外，如果您面對的是一個政府，經常因為異議分子使用加密工具而將他們監禁，那麼使用更簡單的技巧 (比如一套預先安排的代碼)，而非留下證據證明您在筆電上使用加密軟體，可能較有幫助。

鑑於以上所述，在下載、購買或使用工具之前，您可以想一些相關問題。

它有多公開透明？Anchor link

儘管數位安全似乎主要與保守祕密有關，但安全研究人員堅信，開放性和透明度會促成更安全的工具。

數位安全社群所使用和推薦的許多軟體都是開源的，也就是說定義其工作原理的代碼是公開可供其他人檢查、修改和共享的。藉由公開這些程式如何運作，這些工具的開發人員邀請其他人去尋找安全漏洞，並幫助改善程式。

開源軟體可能提供更好的安全性，但無法保證。開源的優勢部分依賴於技術專家社群實質檢查程式碼，對於小型 (甚至流行、複雜的) 專案可能難以實現。考慮使用工具時，請看其原始碼是否可取得，以及是否具有獨立的安全審核以確認其安全品質。軟體或硬體至少應該有一份詳細的技術說明，解釋它是如何運作的，供其他專家檢查。

開發人員對於軟體的優缺點有多清楚？Anchor link

沒有軟體或硬體是完全安全的。那些誠實對待產品限制的開發人員或者銷售商會讓您更能確認他們的應用程式是否適合您使用。

不要相信那些聲稱程式碼是「軍用級」或「防國安局」的各種說詞；這些沒有任何意義，而且強烈暗示開發人員可能過於自信或不願意考慮其產品可能存在的缺陷。

由於攻擊者總是試圖找出破壞工具安全的新方法，因此軟體和硬體通常需要更新以修復新的漏洞。如果一個工具的開發人員不願意這樣做，那麼問題可能很嚴重，要不是因為他們害怕負面宣傳，或者是因為他們沒有建立基礎設施來解決問題。

未來無法預測，但是對於工具開發商未來的行為，他們過去的活動會是個很好的指標。如果該工具的網站上面列出了以前的問題以及定期更新和資訊的連結 (特別是自軟體上次更新以來已經過了多長時間)，那麼您可以更加確信他們將來會繼續提供這項服務。

如果開發人員遭到入侵，會發生什麼事？Anchor link

當安全工具開發商建立軟體和硬體時，他們 (就像您一樣) 必須有一個明確的威脅模型。最好的開發人員將會明確地在文件中說明他們可以防範的攻擊者。

但有一種攻擊者，許多開發商不想考慮：如果他們自己遭到入侵，或決定攻擊他們自己的使用者。例如，法院或政府可能會迫使一家公司放棄個人資料，或者建立一個「後門」，去除他們工具提供的所有保護措施。您可能需要考慮開發人員所在的司法管轄區。例如，如果您的威脅來自伊朗政府，美國公司將可不配合伊朗的法院命令，即使它必須遵守美國的命令。

即使開發人員能夠抵制政府施壓，攻擊者也可能試圖藉由闖入工具開發商自己的系統來攻擊其客戶，進而獲得相同的結果。

最有韌性的工具會將這種攻擊視為可能，並在設計中對其防禦。尋找那些保證開發人員不能存取私人資料，而不是承諾開發人員不會這麼做的說法。尋找以對抗法院命令，保護個人資料著稱的機構。

尋找召回和線上評論 Anchor link

當然，銷售產品的公司和推銷他們最新軟體的熱心人士可能已被誤導、導致誤導，甚至是直截了當的說謊。原本安全的產品未來可能會被發現具有可怕的缺陷。請確保您了解您使用工具的相關最新消息。

您知道還有誰使用相同的工具？Anchor link

一個人要持續追蹤一項工具的最新消息，是很累人的事。如果您的同事使用特定的產品或服務，請與他們合作，以了解所發生的事情。

我們盡量確保本指引中提到的軟體和硬體符合我們上面列出的標準：我們已經盡力，只列出那些對我們目前了解的數位安全有著堅實基礎的產品，對其操作 (及其缺陷) 一般都是透明的，對於開發人員本身可能受到的攻擊有所防範，目前都有進行維護，擁有大量且技術知識豐富的使用者基礎。我們認為在撰寫本文時，有一大批人正在檢查它們的缺陷，很快就會引起公眾的關注。請各位要了解，我們沒有資源對每項產品的安全性進行審查或做出獨立保證，我們不為這些產品背書，也不能保證完整的安全性。

我應該購買哪款手機？哪台電腦？Anchor link

安全培訓人員最常被問到的問題之一是「我應該買 Android 手機還是 iPhone？」或「我應該使用 PC 還是 Mac？」或「我應該用什麼作業系統？」這些問題無法簡單回答。軟體和裝置的相對安全性正在不斷發生變化，因為有新的缺陷出現，舊的漏洞修復。各公司可能會相互競爭，為您提供更好的安全保障，也可能會受到政府的壓力，削弱安全性。

但是，一些普遍的建議幾乎不會錯。購買裝置或作業系統時，請將軟體保持在最新狀態。更新通常會修復舊程式碼中可能用於攻擊的安全問題。較舊的手機和作業系統不再受到支援，即使是安全更新。特別是 Microsoft 已經明確表示，Windows XP 和更早版本的 Windows 將不會收到針對嚴重安全問題的修復。如果您使用 XP，無法期待它能安全不受攻擊。(對於 10.7.5 或「Lion」之前的 OS X 也是如此)。
