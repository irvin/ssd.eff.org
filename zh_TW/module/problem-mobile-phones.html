行動電話的問題

手機已經成為無處不在、基本的通訊工具，現在不僅用來打電話，還用來上網、發簡訊和記錄世界的大小事。

不幸的是，手機的設計並不注重隱私和安全。不僅在保護您的通訊方面做得不好，還會讓您暴露在新的監控風險之下，特別是定位追蹤。大多數手機提供給使用者的控制比個人桌上型或筆記型電腦少得多；更換作業系統更加困難，更難調查惡意程式攻擊，更難刪除或更換不想要的套裝軟體，而且更難阻止像行動服務供應商這樣的組織監控您使用裝置的方式。此外，裝置製造商可能會宣布您的裝置已過時，並停止向您提供軟體更新，包括安全修補程式；如果發生這種情況，您可能沒有辦法尋求這些修補。

有些問題可以通過使用第三方隱私軟體解決 - 但有些問題不能。在這裡，我們將介紹一些手機可能助長監控並破壞使用者隱私的情況。

定位追蹤 Anchor link

從行動電話衍生，最嚴重但通常完全看不見的隱私威脅，就是透過它們廣播的訊號，無時無刻洩漏您的行蹤。至少有四種方式可以追蹤個人手機的位置。

1.行動網路訊號追蹤 - 基地台

在所有的現代行動網路中，只要電話開機並在網路上註冊，電信商就可以計算特定使用者的電話所在的位置。這項能力來自行動網路建立的方式，通常稱為三角測量。

其中一種電信商可以做到的方式，是觀察不同基地台從特定使用者的行動電話觀察到的訊號強度，然後計算出電話必定位於何處，才會得到這些觀察結果。電信商計算出使用者位置的準確性取決於許多因素，包括電信商使用的技術，以及他們在一個地區有多少基地台。在城市街區層級大部分準確，但是在部分系統，準確度可以更高。

只要您的手機開機並向電信商網路傳輸訊號，就無法避免這種追蹤。儘管通常只有行動服務供應商本身才能進行這種追蹤，但政府可能會迫使電信商交出使用者的位置資料 (即時或歷程記錄)。2010 年，一個名叫 Malte Spitz 的德國隱私倡議人士援引了隱私法規，讓他的行動服務供應商交出關於他的記錄；他選擇把這些記錄當作教育資源發布，讓其他人可以瞭解行動服務供應商如何以這種方式監控使用者。(您可以在這裡看看電信商對他掌握的資訊。)政府獲得這種資料的可能性不是推論的，它已經被美國等國家的執法機構廣泛使用。

另一種相關的政府要求稱為基地台傾印 (tower dump)；在這種情況下，政府要求行動服務供應商提供特定時間在某一地區所有行動裝置的列表。這可以用來調查犯罪，或在一場特定抗議活動找出誰在場。(據報導，烏克蘭政府在 2014 年為此目的使用了基地台傾印，列出進行反政府抗議活動時所有在場的手機使用者。)

電信商還可以相互交換有關裝置當前連線位置的資料。這些資料通常比追蹤匯總多組基地台傾印觀測資料的資料精確得多，但它仍然可以作為追踪單個裝置所使用服務的基礎 - 包括查詢這些記錄，以查找一部當前連到行動網路電話所在位置的商業服務，並將結果提供給政府或私人客戶。(華盛頓郵報報導了這種追蹤資訊的可用性。)這種追蹤方法與以前的不同，不涉及強迫電信商交出使用者資料，而是使用已經以商業形式提供的位置資料。

2.行動網路訊號追蹤 - 國際移動用戶識別碼擷取器 (IMSI Catcher)

政府或其他技術高階的組織也可以直接收集位置資料，例如 IMSI Catcher (偽裝的可攜式基地台，以便「捕捉」特定使用者的手機，偵測他們的真實位置及/或監聽通訊)。IMSI 是指識別特定使用者 SIM 卡的國際移動用戶識別碼，儘管 IMSI Catcher 也可以使用裝置的其他屬性來進行鎖定。

IMSI Catcher 需要帶到特定的位置，才能找出或監控在該位置的裝置。目前所有的 IMSI Catcher 都沒有可靠的防禦措施。(有些應用程式聲稱可以檢測它們的存在，但這種檢測並不完美。)在可以停用 2G 支援的裝置上將其停用 (以讓裝置只能連線到 3G 和 4G 網路) 可能會有所幫助，而且如果您不希望在您電信商的服務區域外旅行，則可以停用國際漫遊。這些措施可以防止某些類型的 IMSI Catcher。

3.Wi-Fi 和藍牙追蹤

現代智慧型手機除了行動網路介面之外還有其他無線電發射器。它們通常也支援 Wi-Fi 和藍牙。這些訊號以比行動網路訊號更低的功率傳輸，而且通常只能在短距離內 (例如同一房間或同一建築物內) 接收，但有時使用高階的天線能從非常長的距離外偵測到這些訊號；在 2007 年的一次示威活動中，委內瑞拉的一位專家在幾乎沒有無線電干擾的農村地區，從 382 公里 (237 哩) 外接收到一個 Wi-Fi 訊號。這兩種無線訊號都包含裝置的唯一序號，稱為 MAC 位址，任何能接收訊號的人都可以看到。裝置製造商在裝置製造時選擇此地址，並且不能使用當前智慧型手機附帶的軟體進行更改。

不幸的是，即使裝置沒有主動連線到特定的無線網路，或者即使它沒有主動發送資料，也可以在無線訊號中觀察到 MAC 位址。無論何時在一般的智慧型手機上開啟 Wi-Fi，智慧型手機都會傳輸包含 MAC 位址的訊號，讓附近的其他人識別出該特定裝置存在。這已經用於商業追蹤應用程式，例如讓店家瞭解特定顧客多久光顧一次，以及他們在店裡花費多長時間的統計資訊。截至 2014 年，智慧型手機製造商已經開始意識到這種追蹤的問題，但是這種追蹤方式還會在所有裝置上存在很久，不會修復——如果真有要修復的話。

與 GSM 監控相比，這些追蹤形式對於政府監控來說並不一定有用。因為這些方式在短距離內效果最好，需要事先知道或觀察來確定特定人士的裝置中有哪些 MAC 位址。然而，這些追蹤形式可以高度精確地提示一個人何時進入和離開建築物。關閉智慧型手機上的 Wi-Fi 和藍牙功能可以防止這種類型的追蹤，但這對於經常使用這些技術的使用者來說可能不方便。

Wi-Fi 網路電信商也可以看到每個加入網路裝置的 MAC 位址，這表示他們可以識別特定的裝置，並分辨您是否曾經加入過該網路 (即使您不在任何地方輸入您的姓名或電子郵件位址，或登錄任何服務)。

在少數裝置上，MAC 位址確實可能更改，讓其他人無法在一段時間內輕易識別您的 Wi-Fi 裝置；在這些裝置上，只要使用適當的軟體和配置，就能以例如每天一次的頻率選擇一個全新、不同的 MAC 位址。在智慧型手機上，這通常需要特殊的軟體，如 MAC 位址更換應用程式。目前，這種方式不適用於大多數的智慧型手機。

4.從應用程式和瀏覽網路洩漏位置資訊

現代智慧型手機為手機提供了確定自己位置的方法，通常使用 GPS，有時使用由位置服務公司提供的其他服務 (通常要求這類公司根據手機基地台及/或手機在所在地點可看到的 Wi-Fi 列表來猜測手機的位置)。應用程式可以向電話要求此位置資訊，並使用它來提供基於位置的服務，例如顯示您所在位置的地圖。

其中一些應用程式將藉由網路將您的位置傳送給服務提供商，而這反過來又為其他人提供了一種途徑來追蹤您。(應用程式開發人員可能沒有追蹤使用者的動機，但他們最終仍然有能力做到這一點，他們最終可能會向政府或駭客透露使用者的位置資訊。)一些智慧型手機會讓您控制應用程式是否可以找出您的實際位置；一個好的隱私做法是試圖限制哪些應用程式可以看到這些資訊，並至少確保您的位置只提供給您信任、有充分理由得知您位置的應用程式。

在任何情況下，定位追蹤不僅僅是尋找某人現在在哪裡，像在一個刺激的電影追逐場景中，特務正在街頭追趕某人。也可以瞭解某人的活動歷程，或是他們的信仰、活動參與和個人關係。例如，位置追蹤可以用來試圖找出某些人是否在談戀愛，誰參加了某個特定的會議，或者是誰在某場抗議活動，或者試圖找出一名新聞從業人員的祕密線人。

華盛頓郵報在 2013 年 12 月報導了美國國家安全局 (NSA) 的位置追蹤工具，這些工具主要藉由竊聽電話公司的基礎設施，來觀察特定電話何時連接到哪個基地台，收集「全球手機行蹤」的大量資訊。一個名為 CO-TRAVELER 的工具使用這些資料來找出不同人的動作之間的關係 (找出哪些人的裝置似乎一起移動，以及是否有人似乎在跟蹤另一個人)。

將電話關閉 Anchor link

許多人擔心，即使手機不撥打電話，也可以用來監控他人。因此，進行敏感對話的人有時會被要求將手機完全關閉，甚至將電池從手機中取出。

取出電池的建議似乎主要集中在一種惡意程式，這種惡意程式能讓手機按要求關閉 (最後只顯示空白畫面)，同時確實保持開機狀態，能夠監聽對話或悄悄撥打、接收來電。因此，使用者可能會被欺騙，認為他們已經成功關掉手機，其實並沒有。這種惡意程式確實存在，至少對於某些裝置而言，儘管我們幾乎沒有任何關於其運作效果或使用廣泛程度的資訊。

關閉手機有其自身的潛在缺點：如果一個地方的許多人都同時這麼做，那麼這會帶給行動電信商一個訊號，認為某個因素造成他們關閉手機。(這個「東西」可能是電影院裡一部電影開始放映，也可能是飛機在機場起飛，但也可能是一個敏感的會議或談話。)另一種可能減少資訊的方法，是將每個人的手機留在另一個房間，如此一來手機的麥克風就無法偷聽對話。

拋棄式手機 Anchor link

臨時使用後丟棄的手機，通常稱為拋棄式手機。那些試圖避免政府監控的人，有時會時常更換手機 (和電話號碼)，讓識別他們的通訊變得更加困難。他們會需要使用預付卡手機 (不與個人信用卡或銀行帳戶相關聯)，並確保手機和 SIM 卡未以其身分註冊；在某些國家，這些步驟很簡單，而在某些國家則可能有獲得匿名手機服務的法律或現實障礙。

這種技術有許多限制。

首先，只交換 SIM 卡或將 SIM 卡換到另一個裝置的保護極為有限，因為行動網路會同時觀察 SIM 卡及裝置。換句話說，網路電信商知道在哪些裝置中已經使用了哪些 SIM 卡的歷程，並且可以單獨或者同時追蹤兩者。其次，各國政府一直在開發行動位置分析技術，在這種技術中，位置追蹤可以用來產生關於多個裝置是否屬於同一個人的線索或假設。有很多方法可以達成這個目標。例如，分析人員可以檢查兩台裝置是否常一起移動，或者即使它們在不同時間的使用，也常在相同的實體位置上進行移動。

要成功匿名使用電話服務的另一個問題，是人的通話模式往往非常獨特。例如，您可能會定期打電話給您的家人和同事。即使這些人每個都接到來自許多人的電話，您可能是世界上唯一一個時常以相同號碼打電話給他們的人。所以，即使您突然改變了您的號碼，如果您在您所撥打或接聽的通話中恢復了相同的模式，要找出哪一個新號碼是您的就很直截了當。請記住，這種推斷不僅僅是根據您撥打一個特定號碼的事實，而是根據您所撥打的所有號碼組合的獨特性。(其實，The Intercept 報導，一個叫做 PROTON 的祕密美國政府系統正是這麼做，它使用通聯記錄來識別那些使用新電話號碼「與特定目標，以相似方式」撥打電話的人)。另一個例子可以在 Hemisphere FOIA 文件中找到。該文件描述了 Hemisphere 資料庫 (通話歷程記錄的龐大資料庫) 以及運行它的人如何使用一項功能，可以藉由遵循拋棄式手機通話模式的相似性找出關聯。該文件將拋棄式手機稱為「丟棄式手機」，因為他們的使用者會「丟棄」一部並開始使用另一部 - 但是這種情況發生時，資料庫演算法可以將一部手機與另一部手機關聯起來，只要兩者都用於撥打或接聽類似的電話號碼。

總之，這些事實顯示，要有效使用拋棄式手機來躲避政府監視，至少需要：不能重複使用 SIM 卡或裝置；不能同時攜帶不同的裝置；不在使用不同裝置的地方之間建立實體關聯；而且在使用不同裝置時，不撥打或接聽來自同樣對象的電話。(這個清單並不完整；例如，我們沒有考慮手機銷售地點或使用地點的實際監控風險，或軟體識別特定某人聲音，以自動判定誰在特定電話說話的可能性。)

關於 GPS 的注意事項 Anchor link

全球定位系統 (GPS) 可讓世上任何地方的裝置快速準確地找出自己的所在位置。GPS 的運作原理是分析美國政府提供大眾公共服務的衛星訊號。有種常見的誤解，就是這些衛星以某種方式監看 GPS 使用者或得知 GPS 使用者的位置。其實 GPS 衛星只能傳輸訊號，衛星不會收到或觀察到任何來自手機的任何資訊，衛星及 GPS 系統操作人員不知道任何特定使用者或裝置的位置，甚至不知道有多少人在使用該系統。

這是有可能的，因為各個 GPS 接收器 (如智慧型手機內的接收器) 藉由判定來自不同衛星的無線電訊號到達需要多長時間，來計算它們自己的位置。

那麼，為什麼我們說「GPS 追蹤」呢？通常，這種追蹤是藉由智慧型手機上運行的應用程式達成。它們會向手機的作業系統詢問其位置 (藉由 GPS 判定)。然後這些應用程式能夠藉由網際網路將這些資訊傳輸給其他人。還有一些小巧的 GPS 接收裝置，可以藏在某人的物品或車上；那些接收器會判定自己的位置，然後藉由網路 (通常是行動網路) 主動轉發。

監視行動通訊 Anchor link

行動電話網路起初的設計並非使用技術方式來保護使用者的電話不受竊聽。這表示任何有正確類型無線電接收器的人都可以聽到通話。

現在的情況有所好轉，但有時改善程度不大。行動通訊標準中加入了加密技術，以防止竊聽。但是這些技術許多都設計不良 (有時是故意的，由於政府壓力，不能使用強力加密！)。它們部署不均勻，因此它們可能在一個電信商有，另一個電信商沒有；或者在一個國家有，另一個國家沒有；有時則是沒有正確實施。例如在某些國家，電信商根本不支援加密，或者使用過時的技術標準。這表示使用正確類型無線電接收器的人，通常仍然可以攔截藉由無線電傳輸的通話和文字訊息。

就算使用了最佳的產業標準 (一些國家和行動服務供應商有這麼做)，仍然有人可以竊聽。至少，行動服務供應商本身能夠攔截、記錄所有通話、發送訊息的對象、時間、內容等相關資料。這些資訊可能藉由官方或非正式協議，提供給當地或外國政府。在某些情況下，外國政府也侵入了行動服務供應商的系統，以祕密存取使用者的資料。另外，您實體位置附近也可能有人使用 IMSI Catcher (如上所述)。這些裝置可以欺騙您的手機使用他們的假「基地台」，而非您行動服務供應商的合法基礎設施，在這種情況下，操作 IMSI Catcher 的人可能可以攔截您的通訊。

最安全的做法，是假定傳統的通話和文字簡訊不能防止竊聽或錄音。儘管各地、各系統之間的技術細節差別很大，但技術保護通常很薄弱，在許多情況下可以繞過。請參閱與他人溝通，瞭解如何更安全地傳送文字簡訊和交談。

當您使用安全通訊應用程式進行通訊 (無論是藉由語音還是文字) 時，情況會有所不同，因為這些應用程式可以應用加密來保護您的通訊。這種加密可以更強力，提供更有意義的保護。使用安全通訊應用程式進行通訊獲得的保護程度，絕大部分取決於您使用的應用程式以及它們的運作方式。一個重要的問題是通訊應用程式是否使用端對端加密保護您的通訊，以及是否有任何方式讓應用程式開發人員撤銷或繞過加密。

用惡意程式感染手機 Anchor link

手機可能會感染病毒和其他類型的惡意程式 (惡意軟體程式)，原因可能是使用者被騙安裝惡意程式，或者有人能夠利用現有裝置軟體中的安全漏洞入侵裝置。與其他類型的運算裝置一樣，惡意程式可以監視裝置的使用者。

例如，手機上的惡意程式可能會讀取裝置上的私人資料 (如儲存的文字簡訊或照片)。它還可以啟動裝置的感測器 (如麥克風、攝影機、GPS)，以尋找手機位置或監視環境，甚至讓手機故障。

這種技術已經被部分政府用來使用人民的手機監視他們，並且在室內有行動電話時造成了對於進行敏感對話的隱憂。有些人藉由進行敏感對話時將手機拿到另一個房間或關閉電源，來應對這種可能性。(政府本身經常禁止人民，甚至政府人員把個人手機帶入某些敏感的設施，主要是因為擔心電話會被軟體感染，將對話記錄下來。)

另外值得關注的是，惡意程式理論上可使手機假裝關機，同時暗中保持開機 (並顯示黑畫面，以致使用者誤以為手機已關機)。這種擔心已經導致一些人在進行非常敏感的談話時，將電池從裝置上移除。

正如我們上面所討論的，行動服務供應商可以注意到基於關閉手機的預防措施；例如，如果有十個人同時到同一棟大樓，然後全部同時關掉手機，那麼行動服務供應商或者檢視其記錄的人可能會得出這樣的結論：這些人都在同一個會議上，與會者都認為內容敏感。如果與會者將手機留在家中或辦公室，這將更難察覺。

扣留手機進行鑑識分析 Anchor link

行動裝置的鑑識分析是一個非常發達的專業。一位分析專家將把一部扣留的裝置連接到一部特殊的機器上，該機器可以讀出裝置中儲存的資料，包括先前的活動記錄、通話和文字訊息。鑑識分析可能可以回復使用者通常無法看到或存取的記錄，例如遭到刪除的文字訊息，可以反刪除。鑑識分析通常可以繞過簡單的螢幕鎖定形式。

有許多智慧型手機應用程式和軟體功能試圖阻止或防止某些資料和記錄的鑑識分析，或加密資料使分析人員無法讀取。此外，還有遠端抹除軟體，讓手機擁有者或擁有者指定的人將電話根據要求刪除某些資料。

如果您的手機是由犯罪分子帶走的，這個軟體可以有效防止資料遭人取得。但是請注意，故意破壞證據或妨礙調查可能視為單獨的犯罪行為起訴，往往造成非常嚴重的後果。在某些情況下，政府可能更容易證明和考慮進行比原先調查的指控罪行更加嚴重的懲罰。

手機使用模式的電腦分析 Anchor link

政府也有興趣使用電腦分析許多使用者手機的資料，以自動搜尋某些模式。這些模式可以讓政府分析人員找到人民​​以不尋常的方式使用手機的情況，例如採取特殊的隱私防護措施。

一些政府可能試圖從資料分析中發現的事例有：自動判斷某些人是否相互認識；偵測一個人何時使用多部電話或換用電話；偵測一群人何時一起旅行或定期見面；偵測一群人何時以不尋常或可疑的方式使用手機；查明新聞從業人員的機密來源。
