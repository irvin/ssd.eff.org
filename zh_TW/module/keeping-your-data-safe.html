確保您的資料安全

要捍衛您的資料，防範潛在攻擊者的最大挑戰之一，就是您所存放或攜帶資訊的龐大規模，以及從您獲取資訊的難易程度。我們許多人都會在筆記型電腦 (甚至手機) 上傳輸我們的聯絡資訊、通訊和當前文件的完整歷史記錄。這些資料可能包括數十甚至數千人的機密資訊。手機或筆記型電腦可能遭竊，或在一瞬間被人複製資料。

美國只是在邊境扣留和複製資料的許多國家之一。資料可以在攔檢站從您身上拿走、在街上從您身上搶走，或者從您家中偷竊。

正如您可以使用加密讓通訊更安全，您也可以讓那些實際偷竊資料的人更難解密。電腦和手機可以藉由密碼、PIN 碼或手勢進行鎖定，但是如果裝置本身被扣留，這些鎖無法保護資料。繞過這些鎖定相對簡單，因為您的資料在裝置中以易讀取的形式存放。攻擊者所需要做的就是直接存取儲存裝置，在不知道密碼的情況下可以複製或查看資料。

如果您使用加密，您的對手不僅需要您的裝置，還需要您的密碼來解密加密的資料 - 沒有其他捷徑。

這是加密所有資料，而不僅僅是幾個資料夾最安全和最簡單的方法。大多數電腦和智慧型手機都提供完整的全機加密選項。Android 在其「安全性」(Security) 設定下提供，iPhone 和 iPad 等 Apple 裝置將其稱為「資料保護」(Data Protection)，如果您設置了密碼，此項功能就會打開。在運行 Windows Pro 的電腦上，它稱為「BitLocker」。 

BitLocker 的程式碼是封閉的和專有的，這表示外部審閱者很難確切知道它的安全性。使用 BitLocker 需要您信任 Microsoft 提供的加密儲存系統沒有未知的弱點。另一方面，如果您已經在使用 Windows，那麼您已經相當於信任 Microsoft 了。如果您擔心被可能知道、利用 Windows 或 BitLocker 後門的攻擊者監控，可能會想考慮替代的開源作業系統，例如 GNU/Linux 或 BSD，特別是針對安全攻擊強化的版本，如 Tails 或 Qubes OS。

Apple 在 macOS 上提供了一個內建的全機加密功能，名為 FileVault。在 Linux 套件版本上，通常在首次安裝系統時提供全機加密。在本指引更新的時候，我們沒有可以針對不包含 BitLocker 的 Windows 版本推薦的全機加密工具。

不管您的裝置怎麼稱呼加密，它的強度等同於您的密碼。如果您的攻擊者擁有您的裝置，他們就可以費盡全力嘗試新的密碼。鑑識軟體每秒可以嘗試數百萬個密碼。這表示四位數字的 PIN 碼不太可能長時間保護您的資料，甚至較長的密碼也只是稍微拖慢攻擊者的速度。在這些條件下，一個強度非常高的密碼應該要超過十五個字元。

我們大多數人都不會在手機或行動裝置上學習輸入這些複雜密碼。因此，雖然加密可以有效防止偶然存取，但您應該保護真正的機密資料，防止攻擊者實際取得它們，或者保管在更安全的設備上。

建立安全的設備 Anchor link

保持一個安全的環境可能不簡單。最好的狀況是，您必須改變密碼、習慣，也許還有在主電腦或裝置上使用的軟體。最糟的狀況是，您必須不斷思考您是在洩漏機密資訊還是在使用不安全的做法。即使您知道這些問題，一些解決方案可能也不在您的掌握之中。即使您已經解釋了危險，其他人也可能會要求您繼續使用不安全的數位安全措施。例如，同事可能希望您繼續打開他們的電子郵件附件，即使您知道攻擊者可能冒充他們並向您寄送惡意程式。或者您可能擔心您的主電腦已經被盜用。

可以考慮的一項策略，是將有價值的資料和通訊封閉在更安全的電腦上。盡量少用這部電腦，而要使用它的時候，做出行動時要更加注意。如果您需要打開附件或使用不安全的軟體，請在另一部電腦上執行。

如果你正在設立一部安全的電腦，還可以採取哪些步驟來保障安全？

您基本上可以將裝置放在更安全的地方：可以看得出是否被破壞的地方，如上鎖的機櫃。

您可以安裝專注於隱私和安全的作業系統，像是 Tails。您可能無法 (或不想) 在日常工作中使用開源作業系統，但如果您只需要在這部安全裝置存放、編輯和撰寫機密電子郵件，或進行即時通訊，則 Tails 是個好選擇，並且預設為高安全性設置。

一部額外、安全的電腦可能並不如您想像的那樣昂貴。只需要一部很少使用的電腦，運行一些程式，不需要跑得特別快或是全新的。您可以購買一部舊款的小筆電 (netbook)，價格只要現代筆記型電腦或手機的一小部分。舊設備也有優勢，像 Tails 這樣的安全軟體可能比裝在新機型上更適合。

您可以使用這部安全設備來保留機密資料的主要副本。一部安全的設備可以有效保管私人資料，但是您也應該考慮它可能造成的一些額外風險。如果您把最珍貴的資訊集中到這台電腦上，可能會使它成為一個明顯的目標。把這部設備藏好，不要提及它的位置，不要忘記以強式密碼加密硬碟，如果這部設備遭竊，在密碼保險箱的情況下，資料將保持無法讀取。

另一個風險是破壞這部設備將讓您損失唯一的資料副本。

如果您的對手會因為您損失所有資料而受益，不要將它只保存在一個地方，不管那個地方有多安全。製作一個加密副本，並保存在別的地方。

毫不令人意外的是，防範網路攻擊或線上監控的最高層級，就是完全不與網際網路連線。您可以確保您的安全電腦不會連接到本地網路或 Wifi，並且只能使用實體媒體 (如 DVD 或 USB 隨身碟) 將檔案複製到電腦上。在網路安全方面，這稱為在電腦和世界之間的「氣隙」。很少人會做得這麼徹底，但是如果您想保存很少存取但永遠不想失去的資料，這可能是一個選擇。例如一個您只用於重要訊息的加密金鑰 (像是「我的其他加密金鑰已不再安全)，如果您失去聯繫，其他人要如何找到您的密碼或指令列表，或他人委託給您的私人資料備份副本。在大多數情況下，您可能只需要考慮使用隱藏的儲存裝置，而不用完整的電腦。一個藏好的加密 USB 金鑰可能跟斷開網際網路的完整電腦一樣有用 (或無用)。

如果您確實使用安全裝置連線到網際網路，則可以選擇不登錄或使用平常的帳戶。建立另外的網路或電子郵件帳戶，使用此裝置進行通訊，並使用 Tor 將您的 IP 位址對這些服務隱藏。如果有人用惡意程式專門針對您的身分，或者只是攔截您的通訊，另外的帳戶和 Tor 可以幫助您打斷身分與這部設備之間的關係。

另一種關於安全設備的想法是準備一部不安全的設備：一部只有在您進入危險地點或需要冒險操作時使用的裝置。例如，許多新聞從業人員和社運人士在旅行時都會帶上最簡單的小筆電。這部電腦沒有任何文件、平常用的的聯絡或電子郵件資訊，如果被沒收或掃描，不會有任何損失。您可以將相同的策略應用在手機上。如果您通常使用智慧型手機，請考慮購買便宜的一次式或拋棄式手機當作旅行或特定用途的通訊方式。
