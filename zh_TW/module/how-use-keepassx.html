如何：使用 KeePassXC

KeePassX 是一個跨平台的密碼保險箱，讓您在同一個位置儲存您所有的密碼。密碼保險箱不僅可儲存密碼，亦可產生密碼，讓您不用記住密碼，就可在各種網站和服務上使用不同密碼。您唯一要記住的密碼，是解鎖密碼資料庫的主密碼。

許多密碼管理程式和 KeePassXC 的名字很像，例如 KeePassX、KeePass 和 KeePass2。有些使用相同的原始碼，有些則是使用相同的資料庫格式。本指引之所以推薦 KeePassXC 是因為它是一個跨平台的工具，而且比其他程式開發更健全。

密碼保險箱會產生單點故障 (SPOF)，而成為不懷好意的人或敵人明顯的攻擊目標。研究顯示，許多常見的密碼保險箱存在漏洞，因此使用前請特別小心。

下載位置： 

  • Windows/Mac/Linux：https://keepassxc.org/download

電腦需求：Windows 7 或更新的版本，MacOS X 10.7 或更新的版本，Linux (大多數套件版本)

本指引中使用的版本：KeePassXC 2.2.0 (KeePass 程式僅適用於 Windows，KeePassXC 則是跨平台的版本。)

授權：FOSS (主要是 GPLv2)

其他閱讀資訊：https://github.com/keepassxreboot/keepassxc/wiki

級別：初學者

所需時間：只要花 5 分鐘設定，之後就能持續使用一個非常強的強密碼。

KeePassXC 如何運作 Anchor link

KeePassX 使用密碼資料庫，也就是儲存所有密碼的檔案。這些資料庫儲存在電腦硬碟上時會加密，所以如果您的電腦被人偷走且處於關機狀態，對方將無法讀取您的密碼。

密碼資料庫可用主密碼加密。由於主密碼保護您所有其他的密碼，請確保它具備足夠的強度。

使用主密碼 Anchor link

主密碼就像一把鑰匙：要打開密碼資料庫，您必須有正確的主密碼。沒有主密碼，就無法知道密碼資料庫裡面有什麼。使用主密碼保護您的密碼資料庫時，請記住一些事項：

  • 由於主密碼會解密您所有的密碼，所以它必須強度夠強！愈長的主密碼愈難猜，而且長度愈長，您就愈不用擔心要使用特殊字元、大寫字母或數字。請將您的主密碼設計成複雜密碼，它是由一長串對您很好記、對其他人卻非常難記的文字組成的密碼。
  • 您必須能夠記住這個密碼！由於這個密碼將可存取所有其他密碼，因此您需要確保您可以記住它，而不用寫下來。這是使用 Diceware 之類程式的另一個原因 - 您可以使用易於記憶的一般單詞，而不用試著記住符號和大寫字母的不自然組合。

使用金鑰檔案 Anchor link

或者，您可以使用金鑰檔案對您的密碼資料庫進行加密。金鑰檔案的作用方式與密碼相同，每次您要解密時，需要將金鑰檔案提供給 KeePassX。金鑰檔案應儲存在 USB 隨身碟或其他攜帶式媒體上，並且只有在要打開密碼資料庫時才插入電腦。這樣做的好處是，即使有人存取您電腦的硬碟 (也就是您的密碼資料庫)，如果沒有儲存在外部媒體中的金鑰檔案，仍然無法對其進行解密。(此外，對於敵人而言，金鑰檔案可能比正常的密碼更難猜到。)缺點是只要您想存取您的密碼資料庫，您需要有存放金鑰檔案的外部媒體在手邊 (如果您失去或損壞了它，那麼您將無法打開您的密碼資料庫)。

使用金鑰檔案而非密碼是最接近以實體鑰匙打開您密碼資料庫的方式 - 您所需要做的事，不過是插入您的 USB 隨身碟，選擇金鑰檔案，就行了！如果您選擇使用金鑰檔案而非主密碼，仍請確保您的 USB 隨身碟儲存在安全的地方 - 任何發現它的人將能夠打開您的密碼資料庫。

同時使用兩種 Anchor link

加密密碼資料庫的最安全方法，是同時使用主密碼和金鑰檔案。如此一來，您解密密碼資料庫的能力就取決於您所知道 (您的主密碼) 以及您擁有的 (您的金鑰檔案)，而任何想要取得您密碼的惡意實體都需要這兩者。(有了這個說法，請記得您的威脅模型：對於大多數只想儲存密碼的家庭使用者來說，強大的主密碼就足夠了。但是如果您擔心有龐大運算資源的國家級攻擊者，那麼安全性越高越好)。

現在您已經瞭解了 KeePassX 的工作原理，讓我們開始使用它！

KeePassX 入門 Anchor link

一旦您安裝了 KeePassX，將它啟動。啟動後，從檔案 (File) 功能表中選擇「新建資料庫」(New Database)。畫面上會出現一個對話方塊，要求您輸入主密碼及/或使用金鑰檔案。選擇適當的核取方塊。請注意，如果您想看到您正在輸入的密碼 (而不是顯示為黑點)，可以按一下右邊的「眼睛」按鈕。另外，您可以使用任何現有檔案做為金鑰檔案 - 例如，您的貓的照片。您只需要確保您選擇的檔案永遠不會被修改，因為如果它的內容遭到改變，那麼它將能再解密您的密碼資料庫。另外請注意，有時在另一個程式中打開檔案，可能就會修改它；最好的做法是不要打開檔案，除非用來解鎖 KeePassX。(雖然移動或重新命名金鑰檔案是安全的。)

成功初始化密碼資料庫後，應從檔案 (File) 功能表中選擇「保存資料庫」(Save Database) 進行儲存。(注意，如果您願意的話，稍後可以將密碼資料庫檔案移動到硬碟上的任何位置，或者移動到其他電腦上 - 您仍然可以使用 KeePassX 和您之前指定的密碼/金鑰檔案開啟它。)

管理密碼 Anchor link

KeePassX 允許您將密碼分為「群組」(Group)，基本上只是檔案夾。您可以藉由功能表列中的「群組」(Group) 功能表或右鍵按一下 KeePassX 視窗左側窗格中的群組 (Group) 來建立、刪除或編輯群組或子群組 (Subgroup)。將密碼分組不會影響 KeePassX 的任何功能 - 它只是一個方便的管理工具。

儲存/產生/編輯密碼 Anchor link

建立一個新的密碼或儲存您已有的密碼，右鍵按一下要儲存密碼的群組，然後選擇「新增條目」(Add New Entry) (您也可以從功能表列中選擇「條目」(Entries) >「新增條目」(Add New Entry))。基本密碼操作，請執行以下操作：

  • 輸入一個描述性標題，您可以在「標題」(Title) 欄位中識別該密碼條目。
  • 在「使用者名稱」欄位中輸入與此密碼條目相關聯的使用者名稱。(如果沒有使用者名稱，可以留白。)
  • 在「密碼」(Password) 欄位中輸入您的密碼。如果您正在建立一個新的密碼 (例如，如果您正在註冊一個新的網站，而您想建立一個新的獨特隨機密碼)，按一下右邊的「Gen」按鈕。畫面上會出現一個密碼產生器對話方塊，您可以使用它產生一個隨機密碼。在這個對話方塊中有幾個選項，包括密碼要含有什麼類型的字元以及長度。
    • 請注意，如果您產生一個隨機密碼，那麼您不必記住 (甚至不知道) 密碼是什麼！KeePassX 會為您儲存，您隨時都可以將其複製/貼上到適當的程式中。這就是密碼保險箱的重點 - 您可以為每個網站/服務使用不同的長隨機密碼，甚至不需要知道密碼是什麼！
    • 因此，只要服務允許，您的密碼應該盡可能長，並使用各種字元。
    • 等您覺得滿意，請按一下右下方的「產生」(Generate) 以產生密碼，然後按一下「確定」(OK)。產生的隨機密碼將自動輸入到「密碼」(Password) 和「再次輸入」(Repeat) 欄位中。(如果您不是產生隨機密碼，則需要在「再次輸入」(Repeat) 欄位中再次輸入您選擇的密碼。)
  • 最後，按一下確定 (OK)。您的密碼現在儲存在您的密碼資料庫中。要確保更改已保存，請務必藉由前往「檔案 (File) > 保存資料庫 (Save Database)」來儲存已編輯的密碼資料庫。(或者，如果您犯了錯誤，可以關閉然後重新打開資料庫檔案，所有更改都將去除。)

如果您需要更改/編輯儲存的密碼，可以選擇它所在的群組，然後在右側窗格中點兩下其標題，再次彈出「新建條目」(New Entry) 對話方塊。

一般使用 Anchor link

要使用密碼資料庫中的條目，只需右鍵按一下條目並選擇「將使用者名稱複製到剪貼簿」(Copy Username to Clipboard) 或「將密碼複製到剪貼簿」(Copy Password to Clipboard)，然後前往要輸入使用者名稱/密碼的視窗/網站，並將其貼上對應的欄位。(除了右鍵按一下條目，也可以點兩下所需條目的使用者名稱或密碼，使用者名稱或密碼將自動複製到剪貼簿。)

進階使用 Anchor link

KeePassX 最有用的功能之一，是當您按下鍵盤上特殊的按鍵組合時，它可以自動為您輸入使用者名稱和密碼。請注意，雖然此功能只可在 Linux 下使用，其他像 KeePass (KeePassX 的基礎) 這樣的密碼保險箱在其他作業系統上支援這項功能，而且運作原理類似。

要啟用此功能，請執行以下操作。

1.選擇您的全域熱鍵。從「其他」(Extras) 功能表中選擇「設置」(Settings)，然後在左側窗格中選擇「進階」(Advanced)。在「全域自動鍵入捷徑」(Global Auto-Type Shortcut) 欄位中按一下，然後按下您想使用的按鍵組合。(例如按住 Ctrl、Alt 和 Shift 鍵，然後按一下「p」。您可以使用任何您喜歡的組合鍵，但是必須確保它與其他應用程式使用的熱鍵不衝突，所以盡量不要使用 Ctrl + X 或 Alt + F4 等。)對設定滿意後，按一下「確定」(OK)。

2.為特定密碼設定自動鍵入。確認您將視窗打開到您想輸入密碼的地方。然後前往 KeePassX，找到要啟用自動鍵入的條目，然後點兩下該條目的標題以打開「新建條目」(New Entry) 對話方塊。

3.按一下左下角的「工具」(Tools) 按鈕，然後選擇「自動鍵入：選擇目標視窗」(Auto-Type: Select target window)。在出現的對話方塊中，展開下拉方塊並選擇要輸入使用者名稱和密碼的視窗標題。按一下確定 (OK)，然後再按一下確定 (OK)。

測試一下！現在為了對您的使用者名稱和密碼進行驗證，請到您要 KeePassX 為您自動鍵入使用者名稱/密碼的視窗/網站。確保您的滑鼠游標位於您的使用者名稱文字方塊中，然後按一下前面為全域熱鍵選擇的按鍵組合。只要 KeePassX 有開啟 (即使是最小化或非作用中)，您的使用者名稱和密碼應該會自動輸入。

請注意，根據網站/視窗的設置方式，此功能可能無法正確運行。(例如，它可能會輸入使用者名稱，不會輸入密碼。)您可以進行疑難排解並自訂此功能 - 更多資訊，我們建議您在此處查看 KeePass 文件。(雖然 KeePass 和 KeePassX 有一些區別，但是這個頁面應該足以指引您正確的方向。)

建議您使用不易誤按的按鍵組合。您不會想不小心將您的銀行帳戶密碼貼到 Facebook 貼文裡！

其他功能 Anchor link

您可以藉由在搜尋方塊 (KeePassX 主視窗的工具列中的文字方塊) 中輸入內容來搜尋資料庫，然後按 Enter 鍵。

也可以按一下主視窗中的欄標題來排序條目。

您也可以藉由選擇「檔案 (File) > 鎖定工作區 (Lock Workspace)」來「鎖定」KeePassX，這樣您就可以開著 KeePassX，但要求輸入主密碼 (及/或金鑰檔案) 後才能存取您的密碼資料庫。您也可以讓 KeePassX 在靜置一段時間後自動鎖定。這可以防止有人在您離開電腦時存取您的密碼。要啟用此功能，請從功能表中選擇「其他 (Extras) > 設置 (Settings)」，然後按一下安全選項。然後核取「靜置 {數字} 秒鐘後鎖定資料庫」(Lock database after inactivity of {number} seconds) 的方塊。

KeePassX 可以儲存的，不只有使用者名稱和密碼。例如，您可以建立條目來儲存帳號、產品金鑰或序號等其他重要內容。您在「密碼」(Password) 欄位中輸入的資料不一定要是密碼。它可以是任何資訊 - 只要輸入儲存在「密碼」欄位而非真正是密碼的資料 (如果沒有使用者名稱，將「使用者名稱」欄位留白)，KeePassX 就能安全保密地記住它。

KeePassX 是易於使用、功能強大的軟體，我們建議您進行探索，以瞭解它能辦到的所有實用功能。
